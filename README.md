# NYU_DS_group_project

Collaborative group project for the NYU Data Science BootCamp.  
This repository contains analysis, code, and documentation developed by our project team.

## 📌 Project Overview

This project is part of the NYU Data Science BootCamp curriculum.  
Our goals include:

- Applying data analysis and modeling techniques to real-world datasets
- Practicing collaborative coding using Git and GitHub
- Producing reproducible, well-documented code
- Presenting results with clear visualizations and insights
